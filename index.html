📘 사용 가이드

1. 데이터 저장 및 백업

  • ☁️ 로그인하면 데이터가 Firebase에 실시간 저장됩니다.
  • 💾 브라우저에는 동기화 백업 사본이 함께 저장됩니다.
  • 📥 [더보기] 탭에서 JSON 백업/복구가 가능합니다.
  • 💻 같은 Google 계정으로 로그인한 PC와 스마트폰이 동일한 데이터를 공유합니다.

2. 예산 관리

  • 💰 [예산] 탭에서 연간 예산을 카테고리별로 설정하세요.
  • 🚦 80% 사용 시 주의(노랑), 100% 초과 시 경고(빨강) 표시

3. 수입/지출 입력

  • 📝 [내역] 탭에서 수입/지출 내역을 입력합니다.
  • 🧮 금액 옆 계산기 버튼으로 계산 후 자동입력이 가능합니다.

4. 편리한 기능

  • 📑 [더보기] → 전월 복사로 지난달 내역을 복사
  • 🔁 [더보기] → 반복 설정으로 매월 자동 생성
  • 📊 [차트] 탭에서 시각화된 분석을 확인하세요.

5. 앱 설치

  • 📲 [더보기] 탭의 앱 설치 버튼으로 휴대폰 또는 PC에 설치할 수 있습니다.
  • 🍎 iPhone/iPad는 Safari 공유 버튼 → 홈 화면에 추가를 선택하세요.
  • 🔒 설치 기능은 localhost 또는 HTTPS 주소에서 열었을 때 활성화됩니다.
